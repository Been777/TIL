짝수의 합

문제 설명
정수 n이 주어질 때, n이하의 짝수를 모두 더한 값을 return 하도록 solution 함수를 작성해주세요.

제한사항
0 < n ≤ 1000

입출력 예
n	result
10	30
4	6

입출력 예 #1
n이 10이므로 2 + 4 + 6 + 8 + 10 = 30을 return 합니다.
입출력 예 #2
n이 4이므로 2 + 4 = 6을 return 합니다.

풀이

주어진 문제:
주어진 정수 n 이하의 모든 짝수를 더한 값을 반환합니다.
문제 해결 방법:
n 이하의 모든 짝수를 찾아서 더하는 방법을 사용할 수 있습니다.
1부터 n까지의 모든 수를 확인하면서 짝수를 찾아 더합니다.
짝수를 판별하는 방법은 해당 수를 2로 나누었을 때 나머지가 0인지를 확인하는 것입니다.
자바로 구현한 함수:
아래는 주어진 문제를 해결하기 위한 자바 함수입니다.
java
Copy code
public int sumOfEvenNumbers(int n) {
    // 결과값을 저장할 변수를 초기화합니다.
    int sum = 0;

    // 1부터 n까지의 모든 수를 순회하면서 짝수를 확인하고 짝수인 경우에만 합에 더합니다.
    for (int i = 1; i <= n; i++) {
        if (i % 2 == 0) { // 현재 숫자가 짝수인 경우
            sum += i; // 합에 더합니다.
        }
    }

    // 최종 결과값을 반환합니다.
    return sum;
}
함수 설명:
함수는 입력값 n을 받아서 1부터 n까지의 짝수를 모두 더한 값을 반환합니다.
sum 변수를 초기화하고, 1부터 n까지의 모든 수를 순회하면서 짝수를 확인하고 짝수인 경우에만 sum에 더합니다.
최종적으로 sum 변수에 저장된 값을 반환합니다.


정답 코드
class Solution {
    public int solution(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        if (i % 2 == 0) {
            sum += i;
        }
    }
    return sum;
  }
}
